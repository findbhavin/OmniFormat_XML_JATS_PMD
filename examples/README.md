# Example Outputs

This directory contains example outputs generated by the OmniFormat XML JATS PMD pipeline.

## Files Included

### 1. article.xml
- **Format**: JATS XML 1.3 (XSD-Compliant)
- **Description**: Standards-compliant JATS XML without DOCTYPE declaration
- **Use Case**: XSD schema validation, general-purpose XML processing
- **Validation**: Can be validated against JATS XSD schemas
- **Features**:
  - Full namespace declarations (XLink, MathML)
  - xsi:schemaLocation for external validators
  - Complete article structure with front, body, and back matter
  - Proper metadata elements

### 2. articledtd.xml
- **Format**: JATS XML 1.3 with DOCTYPE
- **Description**: PMC-compliant JATS XML with DOCTYPE declaration
- **Use Case**: PMC submission, DTD-based validation
- **Validation**: Compatible with PMC Style Checker
- **Features**:
  - DOCTYPE declaration for DTD validation
  - Identical content to article.xml (except DOCTYPE)
  - PMC submission ready
  - Proper reference structure with xref elements

### 3. article.html
- **Format**: HTML5
- **Description**: Human-readable HTML version with embedded styling
- **Use Case**: Web display, accessibility, preview
- **Validation**: W3C HTML5 compliant
- **Features**:
  - Semantic HTML5 structure
  - Embedded CSS for professional styling
  - Responsive design
  - Print-optimized styles
  - Professional table formatting
  - Proper heading hierarchy
  - Accessible markup

### 4. media/ (Directory)
- **Contents**: Extracted images and figures
- **Formats**: PNG, JPG, SVG
- **Description**: All media files referenced in the document
- **Use Case**: Web display, archival

## Generating These Examples

To generate these outputs from a DOCX file:

```python
from MasterPipeline import HighFidelityConverter

converter = HighFidelityConverter('your_document.docx')
converter.run()
```

All outputs will be generated in `/tmp/output_files/`

## Validation Commands

### Validate JATS XML (XSD)
```bash
xmllint --schema JATS-journalpublishing-oasis-article1-3-mathml2.xsd article.xml --noout
```

### Validate PMC Compliance
```bash
cd pmc-stylechecker
xsltproc --path . nlm-style-5-0.xsl ../examples/outputs/articledtd.xml
```

### Validate HTML
```bash
# Using online validator or:
npm install -g html-validator-cli
html-validator article.html
```

## Standards Compliance

- **JATS**: Version 1.3, compatible with 1.4
- **PMC**: Follows PMC tagging guidelines
- **HTML**: HTML5, W3C compliant

## Usage Notes

1. **For XSD Validation**: Use `article.xml`
2. **For PMC Submission**: Use `articledtd.xml`
3. **For Web Display**: Use `article.html`
4. **For Archival**: Keep all formats
